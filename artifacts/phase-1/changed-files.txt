Phase 1 — liq-features changed files
====================================

Code added:
  src/liq/features/volatility/estimators/formulas.py
    Per-bar variance contributions: ctc_var_contribution,
    parkinson_var_contribution, garman_klass_var_contribution,
    rogers_satchell_var_contribution, gk_yang_zhang_var_contribution,
    yz_overnight_term, yz_open_close_term, yz_rs_term. Every function
    takes log prices per [APPENDIX_FORMULAS] notation.
  src/liq/features/volatility/estimators/windowed.py
    trailing_mean (NaN-skipping rolling mean), yz_k_weight,
    yang_zhang_var (compose overnight + k_yz * open_close + (1-k_yz) * rs).

Code modified:
  src/liq/features/volatility/contracts.py
    VolEstimate / VolComponent value-typed fields changed from
    pd.Series (TYPE_CHECKING) to pl.Series (runtime polars). Aligns the
    contract with the workspace's polars-first convention. See
    artifacts/phase-1/decisions.md item 1.
  src/liq/features/volatility/estimate.py
    Real orchestration body: spec validation (VolSpecError for missing
    rv_spec, invalid window/min_periods, missing columns), PIT
    enforcement (VolPITViolationError when max(valid_from) > asof),
    log-column conversion, per-estimator dispatch, c2c risk-variance
    composition (cont + overnight_gap for YZ target), annualization
    when output_unit requests it, VolEstimate emission.

Tests added:
  tests/test_volatility_formulas.py
    10 golden-fixture tests at 1e-12 tolerance for every formula in
    the registry + the Rogers-Satchell drift sanity check and the GK-YZ
    opening-jump-isolation test.
  tests/test_volatility_windowed.py
    29 tests: trailing-mean NaN-gating contracts, yz_k_weight closed
    form, windowed golden tests at 1e-10 for all six estimators at
    windows {5, 10, 15}.
  tests/test_volatility_estimate.py
    19 integration tests: returns-vol-estimate shape, dispatch across
    all six estimators, spec-validation negative paths, PIT property
    test over Hypothesis pit_violating_frame, scale-invariance property
    test over Hypothesis valid_ohlc_frame for YZ and Parkinson,
    annualized variance / vol output with periods_per_year.
