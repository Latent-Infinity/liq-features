RV-noise gate — liq-features changed files
=====================================

Code added:
  src/liq/features/volatility/rv.py
    Replaces the empty foundation skeleton with the minute-mode RV / BPV
    / JV helpers + the §5.3 RV-noise gate:
    - compute_rv(returns) = Σ r_i² (sum of squared log-returns)
    - compute_bpv(returns) = (π/2) · Σ |r_{i-1}| · |r_i| (jump-robust)
    - compute_jv(returns) = max(RV - BPV, 0) (jump variation, clipped)
    - rv_noise_gate(rv_by_interval, *, price_movement, ratio_threshold=2.0,
      unexplained_threshold=0.5): fires when RV_1m ≥ 2 · RV_5m AND
      RV_1m ≥ 2 · RV_15m AND price_movement² < 0.5 · RV_5m (the
      directional move doesn't explain the inflated RV_1m).

  src/liq/features/volatility/decomposition.py
    Replaces the empty foundation skeleton with the §5.4 derivation layer:
    - estimator_dispersion(estimates) = population std across the
      estimator outputs.
    - derive_quality_flags(estimates, *, dispersion_threshold) →
      tuple[str, ...] applying the §5.4 table:
      • GAP_DOMINATED_VOL when ctc ≥ 4× parkinson
      • INTRADAY_RANGE_DOMINATED_VOL when rs ≥ 4× ctc
      • HIGH_ESTIMATOR_DISAGREEMENT when estimator_dispersion > policy bar
      • CTC_DISAGREES_WITH_RANGE when |log(ctc / range_mean)| ≥ log(4)
    Missing keys tolerated — degraded-fallback dicts don't crash.
